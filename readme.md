# CocktailFinder
##### React Native challenge
### Install
Open terminal and run:
```sh
npm install
```
### Run
```sh
npm start
react-native run-android
```
### Libraries used
* [react-native]
* [typescript]
* [react-navigation]
* [react-navigation-stack]
* [react-native-vector-icons]
* [redux]
* [react-native-linear-gradient]
